# Learning Lab

A productivity tool designed to manage focused study sessions and track learning progress.

## Project Status
- Established the core application class structure using Object-Oriented Programming.
- Configured the main UI container to handle margins and layout scaling.
- [x] Initialized OOP class structure.
- [x] Established master container-based layout.
- [x] Added `CTkOptionMenu` for topic selection.
- [x] Implemented internal state list for topic management.
- [x] Buttons for adding and removing courses
- [x] Added timer for study. Can start/pause, add custom study times for the courses and reset the timer
- [x] Added a one time extra timer

## Requirements
- Python 3.x
- CustomTkinter (see `requirements.txt`)

## Installation and Usage
1. Install the required libraries:
   	pip install -r requirements.txt

2. Run the application
	python3 learning_app.py
